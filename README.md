# Profile Card 

A responsive, accessible Profile Card component built with semantic HTML, CSS (Flexbox), and vanilla JavaScript.

## Features
- Semantic HTML with proper tags (`article`, `header`, `figure`, `nav`, `section`).
- Responsive layout (stacked on mobile, side-by-side on desktop).
- Accessible with keyboard navigation, focus styles, and `alt` text.
- Displays current time in milliseconds using `Date.now()`.
- All required `data-testid` attributes for automated testing.
- Social links open in new tabs with `rel="noopener noreferrer"`.
